# README
-----------------------

Landee is free business landing page built with Twitter Bootstrap framework and Sass CSS preprocessor (with Compass), so you can develop this template easily. It's made for business owner, startup or creator who want to begin or boost their internet-visibility. This template is ultra responsive, clean and easy to customize.
